Este proyecto es una página web de presentación dedicada a la historia y los logros del club de fútbol Colo Colo.

## Descripción

La página está construida con HTML, CSS y Bootstrap 5, y presenta:

- Una sección hero con estilo visual oscuro y elementos gráficos dinámicos.
- Una banda de estadísticas destacando hitos históricos.
- Un sistema de filtros interactivo para explorar logros por década y tipo.
- Un panel de cards con detalles de eventos importantes.
- Una línea de tiempo histórica y un footer temático.

## Archivos

- `colo-colo-historia.html`: Página principal con el contenido completo y estilos inline.
- `README.md`: Información del proyecto.

## Uso

Abrir `colo-colo-historia.html` en un navegador web para ver la página.

## Créditos

Creado como trabajo de sumativa para presentar una página web temática sobre Colo Colo.
